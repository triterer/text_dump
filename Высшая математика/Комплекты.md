Комплекты
========================

tldr: Почти [множества](%D0%9C%D0%BD%D0%BE%D0%B6%D0%B5%D1%81%D1%82%D0%B2%D0%B0.md), но допускают повторения элементов.

Действительно. Записывается примерно так:

Пространство {a, b, c, d}. Коиплекты:
- A = {a, b}
- B = {a, b, c, b, c}
- C = {a, b, b, c, c}
- D = {b, b, b}

Порядок значения не имеет. Значит B = C. Но С называется упорядоченным.

## Членство
Если нам нужно сказать, что некий элемент входит в комплект столько-то раз, то записываем мы это так:

#(b, C) = 2. Читается: элемент b встречается в комплекте C 2 раза.

Если элемент не входит в комплект, то это функция будет равна нулю. В пустой комплект (обознаечается ∅) не входит ни один элемент.

## Мощность
Это сумма членств объектов. Или число элементов в множестве. |B| = 5

## Включение и равенство комплектов
Если состав двух коплектов равен, то комплекты равны. B = C

Если один комплект состоит из элементов другого, но не совпадает. Любой элемент включаемого комплекта по членству меньше или равен членству включающего комплекта. То он является подкомплектом. A ⊆ B

*Строгое включение* - это когда число элементов включаемого комплекта может превышать включающий, но мощность включающего должна быть больше. D ⊂ B

## Операции
- Объединение комплектов A ⋃ B: #(x, A ⋃ B) = max(#(x, A)), #(x, B))
- Пересечение комплектов A ⋂ B: #(x, A ⋂ B) = min(#(x, A), #(x, B))
- Сложение комплектов A + B: #(x, A + B) = #(x, A) + #(x, B)
- Вычитаение комплектов A - B: #(x, A - B) = #(x, A) - #(x, A ⋂ B)

Сложение и объединение коммутативны. Справедливы следующие включения:

A ⋂ B ⊆ A ⊆ A ⋃ B

A - B ⊆ A ⊆ A + B

## Пространство комплектов
Пусть D множество элементов. Тогда мы можем определить $ D^{n} - пространство комлектов, каждый из которых принадлежит D и ни один элемент не входит в какой либо комплект более n раз.

## Отображение Париха
Для конечной области D = {d1, d2, ... dn} существует естественное соответствие между каждым комплектом B над D и n-вектором f={f1, f2, ... fn}, определяемый соотношеием fi = #(di, B). Этот вектор известен как отображение Париха.
