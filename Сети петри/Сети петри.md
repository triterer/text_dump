Сети петри
========================

Это система моделирования систем сотоящих из отдельных *компонент*, которые взаимодействуют между собой. Каждая компонента может быть отдельной системой и иметь свое состояние. Но она обязана быть независимой от других компонент, за исключением случаем взаимодействия с другими компонентами.

Действиям системы свойствены *совмещенность* и *параллелизм*.

Поскольку компоненты системы взаимодействуют, то требуется установление *Синхронизации*.

## Анализ систем
Есть два пути анализа системы с помощью сетей петри.

### Итеративное проектирование системы
1. Используются общепринятые методы проектирования.
2. Затем спроектированная система моделируется сетью Петри
3. Модель сети петри анализируется

На основе анализа делаются выводы и либо цикл перезапускается, либо делается вывод о готовности системы. Хорошим признаком неготовности является проблема при попытке моделирования сети Петри. Схематичное описание выглядит примерно так.

### Моделирование сразу
Система моделируется сразу в терминах сети Петри. Анализ применяется для улучшения самой модели. А задача заключается в самом преобразовании модели в реальую рабочую систему.
